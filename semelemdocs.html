The Semantic Elements

Semantic elements within a document give clear definition of what the tag contains and describes its purpose to the developer and web browser.

<article> | <aside> | <details> | <figure> | <figcaption> | <footer> | <header> | <main> | <nav> | <section> | <summary> | <time>

The <article> Element

The article tag is often confused with the section tag. However, an article contains self-contained content which, when taken out of context, will still make sense to the reader:

Code Result
<html>
<body>
<article>
<h1>Definition</h1>
<p>1. a statement of the exact meaning of a word, especially in a dictionnary.</p>
</article>
</body>
</html>

Definition

1. a statement of the exact meaning of a word, especially in a dictionnary.

The <aside> Element

The aside tag is used to define content aside from the main content of the page. It does however need to be related to the topic on the page:

Code Result
<html>
<body>
<p>I study at CCRI.</p>
<aside>
<h4>CCRI</h4>
<p>Stands for Community College of Rhode Island.</p>
</aside>
</body>
</html>

I study at CCRI.

CCRI

Stands for Community College of Rhode Island.

The <details> Element

The details tag allows the display of additional content which is hidden by default but can be shown if the user whishes to see it:

Code Result
<html>
<body>
<details>
<summary>Click for more content</summary>
<p>Isn't this fun?</p>
</details>
</body>
</html>
Click for more content

Isn't this fun?

The <figure> Element

The figure tag is used to define self-contained content such as images, illustrations, diagrams, etc.. :

Code Result
<html>
<body>
<p>A flower, sometimes known as a bloom or blossom, is the reproductive structure found in plants that are floral (plants of the division Magnoliophyta, also called angiosperms).</p>
<figure>
<img src="flowerImg.jpg" alt="Flower" width="100" height="100">
</figure>
<a href="https://en.wikipedia.org/wiki/Flower" target="_blank">Wikpedia Link</a>
</body>
</html>

A flower, sometimes known as a bloom or blossom, is the reproductive structure found in plants that are floral (plants of the division Magnoliophyta, also called angiosperms).

Wikpedia Link

The <figcaption> Element

The figcaption tag is used within a figure tage in order to add a caption to the figure element:

Code Result
<html>
<body>
<p>A flower, sometimes known as a bloom or blossom, is the reproductive structure found in plants that are floral (plants of the division Magnoliophyta, also called angiosperms).</p>
<figure>
<img src="flowerImg.jpg" alt="Flower" width="100" height="100">
<figcaption><a href="https://en.wikipedia.org/wiki/Flower" target="_blank">Wikpedia Link</a></figcaption>
</figure>
</body>
</html>

A flower, sometimes known as a bloom or blossom, is the reproductive structure found in plants that are floral (plants of the division Magnoliophyta, also called angiosperms).

The <footer> Element

The footer tag defines the footer of a document or a section. It is often used to store information such as copyrights, authorship, other documents belonging to this website, etc.. :

Code Result
<html>
<body>
<footer>
<p>Ⓒ Web page written and design by Dylan Grandjean</p>
<p>Community College of Rhode Island</p>
</footer>
</body>
</html>

Ⓒ Web page written and design by Dylan Grandjean

Community College of Rhode Island

The <header> Element

The header tag defines a section used for introductory content or navigation links. It is placed at the beginning of a document or section:

Code Result
<html>
<body>
<header>
<h1>This is a title</h1>
<p>Here is some content within the header.</p>
</header>
<p>This content is outside of the header.</p>
</body>
</html>

This is a title

Here is some content within the header.

This content is outside of the header.

The <main> Element

The main tag defines the main content of a document:

Code Result
<html>
<body>
<main>
<h1>Most Popular Video Game Genre</h1>
<p>MMORPG, FPS, and RTS are considered three of the most popular genre in video games nowadays.<p>
<article>
<h2>1. Massively Multiplayer Online Role Playing Game (MMORPG)</h3>
<p>MMORPG is a genre played online with large numbers of players, where a player creates a personalized character and assumes their role in a vast interactive world.<p>
</article>
<article>
<h3>2. Fist Player Shooter (FPS)</h3>
<p>First person shooters are a weapon-based combat game played from the first-person perspective, with a camera focusing on the weapon.<p>
</article>
<article>
<h3>3. Real Time Strategy (RTS)</h3>
<p>These games are based around players collecting resources, building units and commanding in-game units into combat.<p>
</article>
<a href="https://blog.gfk.com/2013/04/dominant-video-game-genres-why-are-they-so-appealing/" target="_blank">Article Link</a>
</main>
</body>
</html>

Most Popular Video Game Genre

MMORPG, FPS, and RTS are considered three of the most popular genre in video games nowadays.

1. Massively Multiplayer Online Role Playing Game (MMORPG)

MMORPG is a genre played online with large numbers of players, where a player creates a personalized character and assumes their role in a vast interactive world.

2. Fist Player Shooter (FPS)

First person shooters are a weapon-based combat game played from the first-person perspective, with a camera focusing on the weapon.

3. Real Time Strategy (RTS)

These games are based around players collecting resources, building units and commanding in-game units into combat.

The nav tag allows the user to create a set of navigation links in order to navigate through a page or website:

Code Result
<html>
<body>
<nav>
<a href="#articleTag"><article></a> |
<a href="#asideTag"><aside></a>><br>
<a href="#detailsTag"><details></a> |

<a href="#navTag"><nav></a> |

<a href="#summeryTag"><summary></a> |

The <section> Element

The section tag divides a document into sections such as chapters, header, footer, etc.. :

Code Result
<html>
<body>
<section>
<h1>First Section</h1>
<p>This is the content of section 1.</p>
</section>
<section>
<h1>Second Section</h1>
<p>This is the content of section 2.</p>
</section>
</body>
</html>

First Section

This is the content of section 1.

Second Section

This is the content of section 2.

The <summary> Element

The summary tag defines the visible title of the detail tag which can be seen at all time whether the detail element is opened or closed:

Code Result
<html>
<body>
<details>
<summary>This is the summary tag</summary>
<p>This is the content of the details tag.</p>
</details>
</body>
</html>
This is the summary tag

This is the content of the details tag.

The <time> Element

The time tag allows the page to display a user-readable time or date:

Code Result
<html>
<body>
<p>We celebrate <time datetime="2017-12-25 00:00">Christmas</time> every year.</p>
<p>I wake up at <time>04:30</time> every morning to go to work.</p>
</body>
</html>

We celebrate Christmas every year.

I wake up at 04:30 every morning to go to work.
