The <button> Element

The button element allows users to create a clickable and interactable element which, when programmed to do so, will perform any number of actions.

This is a simple button code, note that, unless some more information is added, this button will not perform any action:

Code Result
<html>
<body>
<button>this is a button</button>
</body>
</html>



By adding a <form> tag, we can now program the button to perform actions using formaction. Note that target="_blank" is not necessary in this case but allows the page to open in a new window or tab:

Code Result
<html>
<body>
<form target="_blank">
<button formaction="https://www.w3schools.com/tags/tag_button.asp">this is a button</button>
</form>
</body>
</html>

For more information, visit W3Schools.com
