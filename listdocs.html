The List Elements

You may want to build lists, numbered or ranked items, or other ordered or unorderd systems. HTML makes it easy to create such documents with the help of list tags.

The ol and ul tags or used to encapsulate list content. ul will create an unordered list whereas ol will give you an ordered one. Within these list, the li tag is used to define elements of the list:

Code Result
<html>
<body>
<ol>
<li>First ordered element</li>
<li>Second ordered element</li>
<li>Third ordered element</li>
</ol>
<ul>
<li>First ordered element</li>
<li>Second ordered element</li>
<li>Third ordered element</li>
</ul>
</body>
</html>
  1. First ordered element
  2. Second ordered element
  3. Third ordered element
  • First unordered element
  • Second unordered element
  • Third unordered element



Furthermore, ordered lists can be customized by using a type attribute:

Code Result
<html>
<body>
<ol type="1">
<li>First type 1 element</li>
<li>Second type 1 element</li>
<li>Third type 1 element</li>
</ol>
<ol type="A">
<li>First type A element</li>
<li>Second type A element</li>
<li>Third type A element</li>
</ol>
<ol type="a">
<li>First type a element</li>
<li>Second type a element</li>
<li>Third type a element</li>
</ol>
<ol type="I">
<li>First type I element</li>
<li>Second type I element</li>
<li>Third type I element</li>
</ol>
<ol type="i">
<li>First type i element</li>
<li>Second type i element</li>
<li>Third type i element</li>
</ol>
</body>
</html>

The dl tag defines a description list. Each element of the list contains a title (dt) and its description (dd):

Code Result
<html>
<body>
<dl>
<dt>Eggs</dt>
<dd>- 2 twelve packs</dd>
<dt>Milk</dt>
<dd>- 1 gallon</dd>
<dt>Pizza</dt>
<dd>- as many as I can afford</dd>
</dl>
</body>
</html>
Eggs
- 2 twelve packs
Milk
- 1 gallon
Pizza
- as many as I can afford



Thanks to CSS list-style-type, it is possible to change the style of an unordered list's markers:

Code Result
<html>
<body>
<ul style="list-style-type:disc">
<li>First disc element</li>
<li>Second disc element</li>
<li>Third disc element</li>
</ul>
<ul style="list-style-type:circle">
<li>First disc element</li>
<li>Second disc element</li>
<li>Third disc element</li>
</ul>
<ul style="list-style-type:square">
<li>First square element</li>
<li>Second square element</li>
<li>Third square element</li>
</ul>
<ul style="list-style-type:none">
<li>First none element</li>
<li>Second none element</li>
<li>Third none element</li>
</ul>
</body>
</html>
  • First disc element
  • Second disc element
  • Third disc element
  • First circle element
  • Second circle element
  • Third circle element
  • First square element
  • Second square element
  • Third square element
  • First none element
  • Second none element
  • Third none element
