The <h1-h6> Element

The heading tags format the font by changing the fonts' weight and size. There are six different heading tags, <h1> defines the most important heading, and <h6> defines the least important one.

Here is an example of each heading:

Code Result
<html>
<body>
<h1>This is an <h1> heading.</h1>
<h2>This is an <h2> heading.</h2>
<h3>This is an <h3> heading.</h3>
<h4>This is an <h4> heading.</h4>
<h5>This is an <h5> heading.</h5>
<h6>This is an <h6> heading.</h6>
</body>
</html>
