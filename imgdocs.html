The <img> Element

The image tag allows users to display images onto their pages. It is a usefull way to convey ideas, or give a unique look to a website. Furthermore, CSS allows a number of customization for images to make an image stand out from the crowd.

This code will display an image onto the page:

Code Result
<html>
<body>
<img src="flowerImg.jpg">
</body>
</html>



It is also possible to create a link out of an image by encapsulating the image withing an <a> tag:

Code Result
<html>
<body>
<a href="https://www.w3schools.com/html/html_images.asp">
<img src="flowerImg.jpg">
<a>
</body>
</html>

For more information, visit W3Schools.com
