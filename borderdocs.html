The border Property

The border property allows a user to create frames around content. These frames can be further decorated and formatted in order to give each page their unique look and feel. It is recommended to keep CSS and HTML codes in seperate files, however, in this example, the CSS portion will be displayed within a <style> tag:

Borders come in all shape, each of them can be defined through a CSS property:

Code Result
<html>
<body>
<p class="none">This text has no border.</p>
<p class="hidden">This text has a hidden border.</p>
<p class="dotted">This test has a dotted border.</p>
<p class="dashed">This test has a dashed border.</p>
<p class="double">This test has a double border.</p>
<p class="groove">This test has a groove border.</p>
<p class="ridge">This test has a ridge border.</p>
<p class="inset">This test has an inset border.</p>
<p class="outset">This test has an outset border.</p>
<p class="solid">This test has a solid border.</p>
</body>
<style>
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.solid {border-style: solid;}
</style>
</html>

Just like fonts, borders have widths which can be modified:

Code Result
<html>
<body>
<p class="one">This text has some meaning.</p>
<p class="two">This text has some meaning.</p>
<p class="three">This text has some meaning.</p>
<p class="four">This text has some meaning.</p>
</body>
<style>
p.one {border-style: dotted; border-width: 5px;}
p.two {border-style: double; border-width: 2ch;}
p.three {border-style: dashed; border-width: thick;}
p.four {border-style: inset; border-width: 5em;}
</style>
</html>

This text has some meaning.

This text has some meaning.

Life isn't all black and white. Luckily, CSS allows the addition of colors to borders as well in order to create vivid and lively elements which will breath life into a page:

Code Result
<html>
<body>
<p class="blueDot">This is a calming blue border.</p>
<p class="redDash">This is an aggressive red border.</p>
<p class="greenSolid">This is a bright green border.</p>
<p class="yellowOutset">This is a sunny yellow border.</p>
</body>
<style>
p.blueDot {border-style: dotted; border-color: blue;}
p.redDash {border-style: dashed; border-color: #FF0000;}
p.greenSolid {border-style: solid; border-color: rgb(107, 244, 66);}
p.yellowOutset {border-style: outset; border-color: hsl(50, 100%, 63%);}
</style>
</html>

This is a calming blue border.

This is an aggressive red border.

This is a bright green border.

This is a sunny yellow border.



Border can be further customized by creating single bordered text. When defining a border's property for one side only, the width, style, and color can be defined within the same property:

Code Result
<html>
<body>
<p class="first">This is a calming blue border.</p>
<p class="second">This is an aggressive red border.</p>
<p class="third">This is a bright green border.</p>
<p class="fourth">This is a sunny yellow border.</p>
</body>
<style>
p.first {border-left: 5px solid blue; background-color: lightblue;}
p.second {border-top: 10px dotted red; background-color: lightpink;}
p.third {border-right: 10px double green; background-color: lightgreen;}
p.fourth {border-bottom: 1em dashed yellow; background-color: lightyellow;}
</style>
</html>

This is a calming blue border.

This is an aggressive red border.

This is a bright green border.

This is a sunny yellow border.



If square isn't your type, it is possible to round up borders as well by using the border-radius property:

Code Result
<html>
<body>
<p class="square">This is a rectangular border.</p>
<p class="round">This is a slightly rounder border.</p>
<p class="rounder">This is a rounder border.</p>
<p class="roundest">This is the roundest border.</p>
</body>
<style>
p.square {border: 5px solid blue;}
p.round {border: 5px solid blue; border-radius: 5px;}
p.rounder {border: 5px solid blue; border-radius: 8px;}
p.roundest {border: 5px solid blue; border-radius: 12px;}
</style>
</html>

This is a rectangular border.

This is a slightly rounder border.

This is a rounder border.

This is the roundest border.



With all the elements discussed on this page, it is easy to believe everything was covered, however, it is possible to mix these properties in order to give your borders unforgettable and trully personal styles:

Code Result
<html>
<body>
<p class="mix">This is one crazy border!</p>
</body>
<style>
p.mix {
border-style: dotted dashed double solid;
border-width: 2px 5px 10px 15px;
border-color: blue green yellow red
border-radius: 8px;
}
</style>
</html>

This is one crazy border!
