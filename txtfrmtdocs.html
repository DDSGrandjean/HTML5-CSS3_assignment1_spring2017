The Text Formatting Elements

Text elements are tags which format the text they surround in order to give them an importance or meaning. Select one of the tags hereunder in order to be directed to its section.

<b> | <strong> | <i> | <em> | <mark> | <small> | <del> | <ins> | <sub> | <sup>

The <b> Element

The bold tag increases the font's weight of the element within it:

Code Result
<html>
<body>
<p>This is a <b>bold text</b></p>
</body>
</html>

This is a bold text.

The <strong> Element

The strong tag increases the font's weight as well, however, there is a sense of importance to the targeted words when this tag is used:

Code Result
<html>
<body>
<p>This is a <strong>strong text.</strong></p>
</body>
</html>

This is a strong text.

The <i> Element

The i tag is used to set a diferent mood to the words within it. It can be used for side notes, quotes, names, sentences in different languages, etc.. :

Code Result
<html>
<body>
<p>This is an <i>italic text.</i></p>
</body>
</html>

This is an italic text.

The <em> Element

The em tag looks similar to italicized text, however, unlike the i tag, it is used to emphasize its content and give it importance over the rest of the text:

Code Result
<html>
<body>
<p>This is an <em>emphasized text.</em></p>
</body>
</html>

This is an emphasized text.

The <mark> Element

The mark tag defines a highlighted portion of text which allows the element within the tag to stand out to the reader:

Code Result
<html>
<body>
<p>This is a <mark>marked text.</mark></p>
</body>
</html>

This is a marked text.

The <small> Element

The small tag decreases the font's size. It is often used for side text or footer content:

Code Result
<html>
<body>
<p>This is a <small>small text.</small></p>
</body>
</html>

This is a small text.

The <del> Element

The del tag defines deleted text:

Code Result
<html>
<body>
<p>This is a <del>deleted text.</del></p>
</body>
</html>

This is a deleted text.

The <ins> Element

The ins tag defines a portion of text that was inserted and underlines it. This tag is often used along with the del tag in order to insert a new entry after a deleted one:

Code Result
<html>
<body>
<p>This is a <del>deleted</del> <ins>inserted</ins> text.</p>
</body>
</html>

This is a deleted inserted text.

The <sub> Element

The sub tag is used to display smaller characters half a character lower than regular text. It is often use in mathematics and chemistry in order to display subscripts:

Code Result
<html>
<body>
<p>This is a <sub>subscripted text.</sub></p>
</body>
</html>

This is a subscripted text.

The <sup> Element

The sup tag, as oposed to the sub tag, will display the characters half a character above the text. It is often used for mathematical formulas and footnotes :

Code Result
<html>
<body>
<p>This is a <sup>superscripted text.</sup></p>
</body>
</html>

This is a superscripted text.
