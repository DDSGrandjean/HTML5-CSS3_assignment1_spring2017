The color Property

The color property in CSS allows the user to customize the color of content. It is recommended to keep CSS and HTML codes in seperate files, however, in this example, the CSS portion will be displayed within a <style> tag:

In order to change the color of an element, the tag needs to be referenced which means that every element which is placed within a particular tag will be affected by the styling property:

Code Result
<html>
<body>
<h1>This header is GREEN</h1>
<p>This paragraph is RED</p>
</body>
<style>
h1 { color: green;}
p { color: #DC143C}
</style>
</html>

This header is GREEN

This paragraph is RED
