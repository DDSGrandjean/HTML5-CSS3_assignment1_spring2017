The background Property

A white background can be boring or tiring on the eye. CSS offers the ability to change the color or look of any background through the help of a few different properties. It is recommended to keep CSS and HTML codes in seperate files, however, in this example, the CSS portion will be displayed within a <style> tag:

The simplest way to change a background is by simply changing its color:

Code Result
<html>
<body>
<p>This content has a pink background.</p>
</body>
<style>
body {background-color: lightpink;}
</style>
</html>

This content has a pink background.



It is also possible to use an image as the background of a page:

Code Result
<html>
<body>
<p>This content has a background image.</p>
</body>
<style>
body {background-image: url("flowerImg.jpg");}
</style>
</html>

This content has a background image.



In some cases, it might be necessary to repeat an image:

Code Result
<html>
<body>
<p>This is an unaltered background.</p>
</body>
<style>
body {background-image: url("gradient.png");}
</style>
</html>

This is an unaltered background.



However, as can be seen above, some background only work when repeated one way:

Code Result
<html>
<body>
<p>This is an x-repeating background.</p>
</body>
<style>
body {background-image: url("gradient.png"); background-repeat: repeat-x;}
</style>
</html>

This is an x-repeating background.
